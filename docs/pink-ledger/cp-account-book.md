# 情侣记账本 💕

> 让爱情与理财同行，用数据记录甜蜜时光

## 项目介绍 ✨

**情侣记账本**是一个专为情侣设计的智能记账应用。它不仅仅是一个简单的记账工具，更是情侣间分享生活、规划未来的甜蜜伴侣。

---

## 技术实现 🛠️

### 前端技术栈
```
uni-app           - 跨平台应用开发框架
Vue 3             - 渐进式框架，组件化开发
uni-icon          - uni-app 图标组件库
```

### 数据存储
```
LocalStorage     - 浏览器端本地存储
SQLite           - 轻量级数据库
```
