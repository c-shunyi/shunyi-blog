# 情侣记账本 💕

> 让爱情与理财同行，用数据记录甜蜜时光

## 项目介绍 ✨

**情侣记账本**是一个专为情侣设计的智能记账应用。它不仅仅是一个简单的记账工具，更是情侣间分享生活、规划未来的甜蜜伴侣。

## 功能特色 🌟

### 智能记账（实现）
- **快速记账** - 语音记账、拍照记账、一键记账
- **分类管理** - 生活用品、约会消费、礼物支出等情侣专属分类

### 情侣模式（未开发，计划中）
- **共享账本** - 两人共同管理一个账本，透明理财
- **角色权限** - 可设置记账权限和查看权限
- **爱心统计** - 为对方花费金额统计，用数据表达爱意
- **纪念日提醒** - 恋爱纪念日、生日等重要日子消费提醒

### 数据分析（未开发，计划中）
- **消费趋势** - 月度/年度消费趋势分析
- **分类占比** - 各类消费占比饼图
- **情侣对比** - 两人消费习惯对比分析
- **预算跟踪** - 预算执行情况实时跟踪

### 目标管理（未开发，计划中）
- **共同目标** - 设置旅行基金、买房基金等共同目标
- **存钱挑战** - 52周存钱法、365天存钱法等趣味挑战
- **进度可视化** - 目标完成进度条和里程碑奖励

## 技术实现 🛠️

### 前端技术栈
```
uni-app           - 跨平台应用开发框架
Vue 3.x           - 渐进式框架，组件化开发
uni-ui            - uni-app 官方 UI 组件库
uni-icon          - uni-app 图标组件库
```

### 数据存储
```
uni-storage      - uni-app 本地数据存储
uniCloud         - uni-app 云服务（计划中）
LocalStorage     - 浏览器端本地存储
Data Backup      - 数据备份与恢复
```
